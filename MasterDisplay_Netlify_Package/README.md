# Master Display: GDR Tools — Netlify package

## Come pubblicare su Netlify (drag & drop)
1. Scarica questo zip e **scompattalo**.
2. Vai su https://app.netlify.com/drop e **trascina la cartella**.
3. Carica nella sottocartella `music/` il tuo file **masterdisplaymusic.mp3** se vuoi la musica di sottofondo.
4. Apri l'URL generato da Netlify. In Safari su iOS puoi usare **Condividi → Aggiungi alla Home** per installarla come app.

## Contenuto
- `index.html` – app completa (mappe, dadi, iniziativa)
- `manifest.webmanifest` – PWA
- `service-worker.js` – cache offline base
- `icons/` – icone 192px e 512px
- `music/` – metti qui `masterdisplaymusic.mp3`
